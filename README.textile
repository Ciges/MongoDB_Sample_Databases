Para cargar la base de datos "employees" en MongoDB habra que ejecutar los siguientes comandos:

<code>
$ git clone git@github.com:Ciges/MongoDB_Sample_Databases
$ cd MongoDB_Sample_Databases/mongodb
$ cd dump
$ tar -xjvf employees.tar.bz2
$ mongorestore employees
</code>

Una vez hecho esto deberiamos tener una base de datos "employees" con dos colecciones "departments" y "employees" de 9 y 300.024 elementos respectivamente

<code>
$ mongo
MongoDB shell version: 2.4.8
connecting to: test
> use employees
switched to db employees
> show collections
departments
employees
system.indexes
> db.employees.count()
300024
> db.departments.count()
9
</code>
